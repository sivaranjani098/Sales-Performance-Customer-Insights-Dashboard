# Sales-Performance-Customer-Insights-Dashboard

This project presents an interactive Power BI Sales Dashboard designed to analyze sales performance, revenue trends, customer behavior, and product profitability to support data-driven business decisions.

# Project Overview

This project focuses on

- Tracking overall sales performance

- Understanding product and category trends

- Analyzing regional sales contribution

- Identifying top customers and product demand

- Monitoring orders, quantity sold, and profitability

# Dataset

The dataset includes transactional and business-related sales data such as:

1. Order Date

2. Customer Name / Segment

3. Product Name & Category

4. Sales Amount

5. Quantity Sold

6. Profit

# Tools & Technologies

1. Power BI – Data visualization & dashboard design

2.Power Query – Data transformation

3. DAX – Calculated measures and KPIs

4. Excel / CSV Dataset – Data source
